# Testausdokumentti

Sovellusta on testattu automaattisilla yksikkötesteillä.

## Testien suorittaminen

Suorita testit komennolla:
```
poetry run invoke test
```

Generoi testikattavuusraportti komennolla:
```
poetry run invoke coverage-report
```
